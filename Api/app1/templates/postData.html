Projeto Licitação

Este projeto tem como objetivo o desenvolvimento tecnológico de um produto para licitações públicas, baseado em mineração de dados e inteligência artificial. Através do processamento massivo de dados, o principal objetivo é desenvolver robôs capazes de gerar, de forma automática, toda a habilitação necessária para uma empresa participar de uma licitação pública.

Em seguida, fornecer um dashboard com dados para inteligência empresarial com informações de nível de adimplência, estatísticas e dados de licitações anteriores do órgão público-alvo. O projeto é financiado pela Secretaria de Educação Profissional e Tecnológica (SETEC).
